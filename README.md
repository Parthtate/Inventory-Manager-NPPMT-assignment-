# Inventory Manager

A simple inventory management app built with React, Tailwind CSS, and Supabase.

## Tech Stack

- **Frontend**: React + Vite
- **Styling**: Tailwind CSS
- **Backend**: Supabase (PostgreSQL + Auth)
- **Deployment**: Vercel

## Features

- Email/Password authentication
- CRUD operations for products (name, price, quantity, category)
- User-specific data with Row Level Security
- Responsive design
